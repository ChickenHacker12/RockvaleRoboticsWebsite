Changing site settings and content

Getting started

  1. Download and/or open Github Desktop.

  2. Sign in.

  3. Fork and open this repository.

  4. Open "settings.json" in Notepad++, Atom, or another text editor.

  5. Edit and save the file.

  6. Create a new pull request.

  7. Commit and merge.

  8. Done.

Editing settings.json

This is a JSON file. Documentation on JSON can be found here.

  • Use standard HTML5.

  • Replace all quotation marks with apostrophes.

Do not use depreciated HTML elements.

"content": "<center>Some text.</center>"

Not replacing quotation marks will throw errors.

"content": "<p>"Hello," he said.</p>"

Instead try:

"content": "<p>\"Hello,\" he said.</p>" "content": "<a href='https://example.com/'>Link to example site</a>"
